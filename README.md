2️⃣ NumPy Fundamentals Notebook
The numpy_all.ipynb file is a practical resource for learning and practicing the core functionalities of the NumPy library. It serves as a strong foundation for anyone starting with Python for data science.

Topics Covered:
Array Creation: Generating 1D and multi-dimensional arrays efficiently.

Indexing and Slicing: Advanced techniques for accessing and filtering array elements (including boolean indexing).

Vectorized Operations: Demonstrating the speed and efficiency of element-wise arithmetic operations.

Array Attributes: Checking array properties (shape, ndim, dtype).
